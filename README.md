# Marvel React App

> The project was completed in 3 days from an api Marvel. We retrieve the data and display it.

## Site composition

- Login
- Signup
- Characters
- Character/By Id
- Comics
- Bookmark

## Technologies

- axios - version 0.21.2
- React - version 17.0.2
- js-cookie - version 2.2.1
